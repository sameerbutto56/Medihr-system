# Medihr-system

MediHR is a comprehensive hospital and HR management system.

## Features

- **HR Portal**: Employee management, payroll generation, attendance tracking, and performance analytics.
- **Hospital Portal**: Patient registration, medical records, appointment scheduling, and billing/invoicing.
- **Role-based Access**: Separate dashboards for CEOs, HR personnel, and Employees.
- **Real-time Data**: Integrated with Firebase for live updates and data persistence.

## Technologies

- React
- Vite
- Firebase
- Recharts
- Tailwind CSS (Vanilla CSS used for custom styling)
- Lucide React (Icons)

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
